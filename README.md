# dotfiles
- configuration dotfiles repository

## Initial setting

```
# install GNU GLOBAL
## Mac
$ brew install global

# install peco
## Mac
$ brew install peco

# clone repository
$ git clone --recursive git@github.com:kawakawaryuryu/dotfiles.git

# execute script
$ ./link.sh

# install vim plugin (on vim)
:call dein#install
```
